# readme - sistema de autenticação com JSON

Neste sistema utilizei a biblioteca System.Text.Json, melhorando mais ainda a lógica de serialização e desserelização de arquivos do tipo JSON. O sistema conta com um loop foreach para percorrer a lista de objetos do tipo "Usuario", e também métodos específicos para cada função no menu.

Nas próximas vezes, há grandes chances de eu fazer a implementação deste sistema no exercicio 06 (banco legacy), para que os clientes do banco possam efetuar transações entre as contas e também fazer autenticação para poder entrar na sua conta.

Isso reforça mais ainda a lógica POO da abstração, e também como percorrer lista de dados.
Estudar JSON tem sido um grande passo à frente, pois desta forma posso tratar dados de forma mais real e não-volátil.
