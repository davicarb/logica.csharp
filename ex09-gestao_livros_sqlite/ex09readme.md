## exercicio 09 - utilizando database sqlite (usando biblioteca using Microsoft.Data.Sqlite) para permanência de dados.
## para manipulação dos dados, foi utilizado ado.net

* neste exercício pratiquei, pela primeira vez, o CRUD básico do SQL (neste caso, SQLite):
- Create;
- Read;
- Use;
- Delete.

* note que no exercício foi utilizado SQLite, uma variação do SQL (structured query language), onde o arquivo database permanece no computador do usuário, e não em um servidor hospedado.

* nele, tive o primeiro contato com a linguagem do SQL, e como é possível manipular dados numa database usando ado.net, uma forma mais "low-level" de modificar e ler os dados.

  apesar de mexer nos dados de uma maneira mais low-level com a biblioteca de classes ado.net, é possível, através dela, ver como realmente os dados são armazenados nas tabelas SQL, e como elas realmente funcionam. desta forma eu posso aprender como realmente funciona o SQL, que é um banco de dados relacional.
